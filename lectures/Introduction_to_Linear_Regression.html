Introduction to Linear Regression

A Linear Model

  • Linear regression is a common technique in the social sciences.
  • Distills complex relationships to a simple linear relationship.

\[y_i = \alpha + \beta x_i\]

  • Intercept (\(\alpha\)): The point at which the regression line crosses the \(y\)-axis, or the value of \(y\) when \(x=0\).

  • Slope (\(\beta\)): The change in \(y\) for every unit change in \(x\). Formally: \(\frac{\partial y}{\partial x} = \beta\)

Example

\[y_{\text{RepublicanCandidate}} = \alpha + \beta_1 x_{\text{PID}}\]

Some Common Misconceptions

Reverse Causality

\(x\) may be caused by \(y\) rather than the reverse.

Confounding

\(x\) may be related to \(y\), but there is a common variable affecting both.

Mediation

A relationship may be indirect through another variable.

The Challenge

It is difficult to sort these out using a regression model with observed cross-sectional data.

In most applications, we observe the outcome of a process. It is often difficult to make claims about the process itself.

Observation versus Manipulation

Experimental versus Observational

Is \(x\) observed or under the control of the researcher?

  • If observational: We can examine whether a relationship exists, but that relationship does not imply causation.

Correlation versus Regression

The two are intimately related, but we make several key assumptions in the regression equation.

  • Dependent variable (regressand): The outcome
  • Explanatory variable (regressor): The predictor
  • The explanatory variable is fixed and exogenous; the dependent variable is random and endogenous (Gujarati and Porter, p. 21)

Randomization and Control

  • In some cases, \(x\) can be manipulated through random assignment.
  • Assign units to a treatment or control condition.
  • Gold standard: Random assignment to treatment conditions.

Theoretical Advantage

When units are randomly assigned to treatment: \[\text{cor}(x, \epsilon) = 0\]

This is because confounding variables are theoretically unrelated to treatment assignment.

This is a key assumption of the regression model that experiments naturally satisfy.

Treatment Effects

Let \(T_i\) represent assignment to a treatment condition:

\[y_i = \alpha + \beta T_i + \epsilon_i\]

Here, \(\beta\) is the treatment effect, the Average Treatment Effect (ATE):

\[E(\beta) = E(\delta_i) = E(Y_{i,1} - Y_{i,0}) = E(Y_{i,1}) - E(Y_{i,0})\]

Important Note

If the true model is \(y_i = \alpha + \beta T_i + \omega x_i + \epsilon_i\) and \(X\) and \(T\) are correlated, then: \[\text{cor}(x, \epsilon) \neq 0\]

Random assignment to \(T\) theoretically ensures \(\text{cor}(x, \epsilon) = 0\) because confounders are equally distributed across treatment groups.

Key Terms and Definitions

Data Types

Random/Stochastic Variables

Variables that take on a set of values drawn from a probability distribution.

Cross-Sectional Data

Units observed once. Indexed with \(i\): \(x_i\)

Time Series Data

A single unit observed multiple times. Indexed with \(t\): \(x_t\) - May trend (systematic change) or be stationary (randomly varying around a mean)

Panel Data

Multiple units observed over multiple time periods. Indexed with \(i\) and \(t\): \(x_{it}\)

A Linear Model (Deterministic)

Assume a deterministic linear relationship:

\[\begin{bmatrix} y_1 &= \alpha + \beta x_1 \\ y_2 &= \alpha + \beta x_2 \\ y_3 &= \alpha + \beta x_3 \\ y_4 &= \alpha + \beta x_4 \\ \vdots & \\ y_n &= \alpha + \beta x_n \end{bmatrix}\]

Example: \(y_{\text{ChildCare}} = 100 + 2 \times X_{\text{Hours}}\)

This is a deterministic relationship: values of \(y\) are perfectly predicted from \(x\).

But…

Do we really have zero error?

The Problem: Omitted Variables

Consider a family working 40 hours per week.

Deterministic model predicts: \(\hat{y} = 100 + 2(40) = 180\)

But other factors matter: - Education level - Geographic region - Family structure - Household composition - Number of children - Childcare type preferences

These differences create variation among families with the same work hours.

Adding the Error Term

We need to include a stochastic component:

\[y_i = \underbrace{\alpha + \beta x_i}_{\text{Deterministic}} + \underbrace{\epsilon_i}_{\text{Stochastic}}\]

For all observations:

\[\begin{bmatrix} y_1 &= \alpha + \beta x_1 &+ \epsilon_1 \\ y_2 &= \alpha + \beta x_2 &+ \epsilon_2 \\ y_3 &= \alpha + \beta x_3 &+ \epsilon_3 \\ y_4 &= \alpha + \beta x_4 &+ \epsilon_4 \\ \vdots & & \\ y_n &= \alpha + \beta x_n &+ \epsilon_n \end{bmatrix}\]

For a particular value of \(x\), the conditional probability distribution is:

\[p(Y | x) = f(x)\]

Or with multiple predictors: \(p(Y | x_1, \ldots, x_k) = f(x_1, \ldots, x_k)\)

where \(p(Y | x)\) is the probability density of \(y\) for a given value of \(x\).

Conditional Distributions

The key insight: for each value of \(x\), there is a distribution of \(y\) values.

  • The mean of this conditional distribution is \(E(Y|x)\)
  • The variance describes how spread out the distribution is
  • Understanding these distributions is fundamental to regression

Key Assumptions of the Linear Model

Normality

The error term is normally distributed: \(\epsilon \sim N(0, \sigma^2)\)

Homogeneity of Variance (Homoskedasticity)

The variance of \(y\) is constant across all values of \(x\)

Linearity

The conditional expectation of \(Y\) given \(x\) is a linear function:

\[\hat{y} \equiv E(Y | x) = \alpha + \beta x\]

Implications

\[E(Y) = E(\alpha + \beta x + \epsilon) = \alpha + \beta x + E(\epsilon)\]

Since \(E(\epsilon) = 0\) by assumption:

\[E(Y | x) = \alpha + \beta x\]

These assumptions together imply: \(\epsilon \sim N(0, \sigma^2)\)

Exploring Linearity: Nonparametric Regression

Approach

  1. Generate \(k\) categories (bins) across levels of \(x\) (e.g., \(x < 25\), \(25 \leq x < 50\), etc.)
  2. Calculate \(\bar{y}_k\) for each bin—the conditional mean of \(y\) in category \(k\)
  3. Compare the linear regression line to these conditional means
  4. Deviation suggests either sampling variation or bias

Interpreting Results

If conditional means fall systematically off the regression line, the linear assumption may be violated.

Sample and Population Regression Functions

Housekeeping: Population vs. Sample

Population Regression Function (PRF)

The true relationship in the population: \[y_i = \alpha + \beta x_i + \epsilon_i\]

Where \(\alpha\), \(\beta\), and \(\epsilon\) are population parameters.

Sample Regression Function (SRF)

Our estimate based on a sample: \[y_i = a + b x_i + e_i\]

Where \(a\), \(b\), and \(e\) are sample statistics (or \(\hat{\alpha}\), \(\hat{\beta}\), and \(\hat{\epsilon}\)), representing predictions of population parameters.

Error Terms: Stochastic vs. Residual

Stochastic Error (Population)

The error in the true population model:

\[y_i - \underbrace{(\alpha + \beta x_i)}_{\text{Population prediction}} = \epsilon_i\]

Residual Error (Sample)

The error in our fitted sample model:

\[y_i - \underbrace{(a + b x_i)}_{\text{Sample prediction}} = e_i\]

Estimation Strategy

Ordinary Least Squares (OLS) minimizes residual error:

\[\min \sum_{i=1}^n e_i^2 = \min \sum_{i=1}^n (y_i - a - b x_i)^2\]

This is the general logic underlying OLS estimation.

Exploring Your Data

Univariate Distributions

Histograms

A useful way to explore univariate properties of a variable:

  1. Generate \(m\) bins
  2. Count observations within each bin
  3. Critical decision: Bin width drastically affects the observed distribution

Histogram showing the distribution of a variable across different bin widths. The choice of bin width can change the apparent shape.

Density Estimation

Mathematical Foundation

Assume starting value \(x_0\), \(m\) bins with width \(2h\).

An observation \(X_i\) falls in bin \(j\) when: \[x_0 + 2(j-1)h \leq X_i < x_0 + 2jh\]

Probability estimate: \[p(x) = \frac{\sum_{i=1}^n [x_0 + 2(j-1)h \leq X_i < x_0 + 2jh]}{2nh}\]

Moving Window Density Estimation

Instead of fixed bins, use a moving window centered at \(x\):

\[p(x) = \frac{\sum_{i=1}^n [x - h \leq X_i < x + h]}{2nh}\]

This is smoother than traditional histograms.

Nonparametric density estimate using a moving window. Smoother than histograms.

Kernel Density Estimation

A more general formulation using a weight function (kernel):

\[p(x) = \frac{1}{nh} \sum_{i=1}^n K\left(\frac{x - X_i}{h}\right)\]

where: - \(K\) is a weight function (e.g., rectangular, triangular, Gaussian) - \(h\) is the bandwidth - Different kernels produce different smoothness properties

Gaussian Kernel Example

Kernel density estimate using a Gaussian (normal) weight function. Very smooth and suitable for continuous variables.

Quantile-Quantile (Q-Q) Plots

Purpose

Examine whether empirical data match a theoretical distribution.

Interpretation

  • Plot empirical quantiles against theoretical quantiles
  • A roughly straight line (\(\text{intercept} = 0\), \(\text{slope} = 1\)) suggests the data match the theoretical distribution
  • Deviations indicate departure from the theoretical distribution

Example: Testing Normality

Compare 1000 observations to a normal distribution.

Summary and Key Takeaways

Before You Analyze

✓ Explore your data thoroughly before advanced analysis - Examine distributions (histograms, density plots) - Check for outliers and anomalies - Visualize bivariate relationships

✓ Understand your data structure - Cross-sectional, time series, or panel? - What scale are your variables (ratio, interval, ordinal, nominal)?

Critical Assumptions

The linear regression model rests on three critical assumptions:

  1. Normality: Errors are normally distributed
  2. Constant Variance: Homogeneity of variance across \(x\)
  3. Linearity: The relationship is actually linear (not curved)

Violation of these assumptions can lead to: - Biased estimates - Incorrect standard errors - Invalid hypothesis tests - Misleading conclusions

Causality Warning

A key reminder: Linear regression identifies associations, not causal effects.

  • Correlation \(\neq\) Causation
  • Random assignment of the treatment is the gold standard for causal inference
  • With observational data, many rival explanations exist

Careful Interpretation Required

  • What is the model?
  • What is the model NOT?
  • Are assumptions met?
  • What are alternative explanations?
